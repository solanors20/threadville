## Instrucciones

1. Descargar GTK con
```bash
sudo apt-get install libgtk-3-dev
```

2. Ejecutar el comando `make` dentro del directorio del proyecto

3. Correr el proyecto con
```bash
./target/threadville
```

4. Confesiones:
- No hay ambulancias
